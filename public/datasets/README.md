# Practice Datasets

This directory contains downloadable practice datasets for hands-on learning.

## Available Datasets

### Beginner Level

1. **retail-sales-basic.xlsx** - Basic retail sales data for beginners
   - Simple product sales with dates, regions, and amounts
   - Perfect for learning basic Power BI concepts
   - Size: ~1MB

2. **customer-data.csv** - Clean customer information dataset
   - Customer demographics and contact information
   - Good for practicing data transformation
   - Size: ~500KB

### Intermediate Level

3. **sales-multi-source.zip** - Multi-table relational dataset
   - Fact_Sales, Dim_Product, Dim_Customer, Dim_Date tables
   - Star schema ready
   - Excellent for learning data modeling
   - Size: ~5MB

4. **financial-messy.xlsx** - Intentionally messy financial data
   - Contains data quality issues to practice cleaning
   - Merged cells, inconsistent dates, missing values
   - Perfect for Power Query transformation practice
   - Size: ~2MB

### Advanced Level

5. **enterprise-supply-chain.zip** - Complex multi-source dataset
   - Inventory, suppliers, orders, shipping data
   - Real-world complexity
   - Designed for advanced modeling and DAX
   - Size: ~15MB

6. **historical-sales-large.csv** - Large historical dataset
   - 100K+ rows of sales transactions
   - Performance optimization practice
   - Size: ~25MB

## Usage Instructions

1. Download the appropriate dataset for your current lesson
2. Import into Power BI Desktop using "Get Data"
3. Follow along with the lesson instructions
4. Save your work as a .pbix file
